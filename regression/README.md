# Regression Models

This directory contains various regression model implementations from scratch.

## Models

- **Linear Regression**: Simple linear regression with normal equation and gradient descent
- **Multiple Linear Regression**: Extension to multiple features
- **Huber Regression**: Robust regression using Huber loss function

Each model includes:
- `model.py`: Core implementation
- `demo.py`: Usage examples
- `notes.md`: Mathematical details and explanations
